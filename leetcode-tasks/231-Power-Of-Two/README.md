##  Power of Two
Given an integer ```n```, return ```true``` _if it is a power of two. Otherwise, return_ ```false```.

An integer ```n``` is a power of two, if there exists an integer ```x``` such that $n == 2^x$.

#### Example 1:
```cpp
Input: n = 1
Output: true
Explanation: 20 = 1
```

#### Example 2:
```cpp
Input: n = 16
Output: true
Explanation: 24 = 16
```

#### Example 3:
```cpp
Input: n = 3
Output: false
```

#### Constraints: 
- $-2^{31} <= n <= 2^{31} - 1$
